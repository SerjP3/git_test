What have we learnt so far?

    HTML
  1. stands for hyper text markup language
  2. describes the strucuture of a web page
  3. consists of a series of elements
  4. tells the browser how to display content
  5. label pieces of content using tags
    Tags & Elements:
  • !DOCTYPE html defines that the document is an HTML 5 document
  • html element is the root element of an HTML page
  • head element contains meta information about the HTML page
  • title element specifies a title for the HTML page
  • body element contains all the visible contents
  • h1..h6 elements defines headings
  • p element defines a pragraph

    • P.S. Some HTML elements have no content. They are callled empty elements and do not have an end tag: br, hr, etc.
